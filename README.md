core-extensions
==========

Collection of simple Ruby core extensions.

Installation
==========

In your Gemfile:

`gem 'core-extensions'`

Install gem manually:

`$ gem install core-extensions`

Run the tests
==========

`$ bundle`

`$ bundle exec rake`
